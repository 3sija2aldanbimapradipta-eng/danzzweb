Ini adalah doumentasi saya saat bertanding. Saya berlatih voli mulai tahun 2023. Saya mulai tertarik bermain voli setelah melihat bapak-bapak lomba voli antar RT, dalam rangka memperingati 17 Agustus. Posisi saya di voli adalah menjadi seorang tosser atau setter, tugas seorang setter adalah mengatur serangan tim dengan mengumpankan bola kepada penyerang(spike), tidak cuman itu setter juga ikut membantu pertahanan tim melalui block dan deffend. Setter juga dapat melakukat serangan yaitu melalui tipuan mapun spike, tapi untuk setter jika ingin sike harus menunggu mendapat kesempatan, karena tidak semestinya seoraqng setter nyemes bola.
Hoby saya adalah berolahraga. Hampir semua olahraga saya kuasai seperti Basket, Sepakbola, Voli, Renang, Lari, Badminton, Futsal,dan gym.

Untuk saat ini saya aktif berolahraga antara lain adalah:

  1. Voli
  2. Gym
  3. Lari

Ini adalah foto saya ketika sedang nge gym. Saya mulai aktif di gym pada tahun 2024 saat saya menduduki bangku kelas 9. Alasan saya nge gym adalah untuk meningkatkan power, dan kekuatan pada otot, karena di permainan bola voli power itu sangat penting. Program gym untuk pemain voli sangat berbeda dengan orang-orang gym pada umumnya, jika orang-orang gym pada umumnya mempunyai program untuk membesarkan otot, beda halnya dengan orang pemain voli. Pemain voli mempunyai program untuk meningkatkan power atau daya ledak tubuh, bukan membasarkan otot. Jika pemain voli mempunyai otot yang besar tubuhnya akan kaku.
Dan yang terakhir ini adalah dokumentasi hasil lari saya ketika sedang mengisi waktu luang dan tidak ada jadwal latihan voli. Lari bukan hanya sekadar aktivitas fisik, tetapi telah menjadi hobi yang memberikan banyak manfaat bagi kesehatan tubuh dan pikiran. Saya mulai menekuni hobi ini karena ingin hidup lebih sehat, dan tanpa disadari, lari telah menjadi bagian penting dalam rutinitas harian saya. Selain meningkatkan kebugaran, lari juga membantu saya mengelola stres dan memberi waktu untuk merenung di tengah kesibukan. Untuk pemain voli lari juga sangat penting karena untuk daya tahan tubuh atau stamina.
